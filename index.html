  • Covid Daily Link

    • After the 'Unlock 1.0' was announced we a group of 3 students were scared that this might worsen the pandemic situation and that fear came true as people have considered it gone and came back to normal lifestyle. So Gagan Chordia , Jinay Bafna and I decided to remind people about the pandemic.   • Teams Clone Link

    • I developed an Android App that can seamlessly perform video calls and messaging between users. User can also set tasks and priority of the task. User Authentication was set up using Firebase Authentication and video Calling was performed using PeerJS. Motivation feature was integrated to keep the user motivated and focused and activity feature shows random activities to user. User database and chat history were managed by Firebase Realtime Database. I have used agile scrum methodology while developing the prototype. I did my work by creating sprints of my own to implement important features in my project. I continuously integrated new features and resolved bugs of the previous integrated features.
